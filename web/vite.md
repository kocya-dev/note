# Vite

https://ja.vitejs.dev/guide/#%E6%A6%82%E8%A6%81

## 環境構築

npm create vite@latest

1. プロジェクト名を決定
2. フレームワークの選択
3. 使用言語の選択 (JS/TS/JS+SWC/TS+SWC)

## 実行

1. 作成したプロジェクトディレクトリに移動
2. npm install (依存モジュールをインストール)
3. npm run dev
4. ブラウザで表示されるアドレスにアクセス

その他
* プロジェクトのビルド
  * npm run build
* ビルドした状態のプレビュー
  * npm run preview
